Objetivos

Esse projeto tem como objetivo estudar e reproduzir a imagem 3 do artigo "A simple theory for the mesopredator release effect: when does an apex predator protect their shared prey from a mesopredator?" de Gaku Takimoto and Shota Nishijima, publicado na Oikos 2022: e09021.

O artigo tem como objeto de estudo o efeito de "liberao do mesopredador", um efeito que ocorre em um sistema predador-mesopredador-presa. Esse efeito ocorre quando h extino do "predador de pice", aquele que no possui predador para si, nesse sistema, causando um desequilbrio, podendo levar ao aumento de mesopredadores e a extino de suas presas compartilhadas, levando enfim, extino do sistema em um efeito cascata.

O sistema considerado um com dinmica representada abaixo.

Modelagem

De acordo com o artigo, o sistema pode ser modelado da seguinte maneira, abseando-se no sistema logstico:

Dinmica da presa compartilhada: $\frac{dP}{dt} = r_P (1 - \frac{P}{K_P}) P - \alpha _PAP - aMP$

Dinmica do mesopredador : $\frac{dM}{dt} = r_M (1 - \frac{M}{K_M}) M - \alpha _MAM - baPM$

Aqui, $P$ a densidade de populao da presa compartilhada, $M$ a densidade do mesopredador, e $A$ a densidade do predador de pice.Os $K$ so a capacidade de suporte de cada populao, $\alpha_P$ a porcentagem de ataque per capita do predador de pice na presa, $\alpha_M$ a porcentagem de ataque per capita do predador de pice ao mesopredador, $r_P$ e $r_M$ so as taxas de crescimento intrnsecas de cada populao, $a$ a taxa de ataque do mesopredador presa, $b$ a taxa de converso de presas capturadas em populao do mesopredador.

Para uma melhor abordagem desse modelo, faamos uma converso de unidades para "enxugar" esse modelo. Primeiro, faamos a seguinte transformao:

$p = \frac{P}{K_P}$

$m = \frac{P}{K_M}$

Que transforma as equaes em:

$\frac{dp}{dt} = r_P (1 - p) p - \alpha _PAp - aK_mpm$

$\frac{dm}{dt} = r_M (1 - m) m - \alpha _MAm - bapmK_P$

Segundo, consideremos agora uma escala de tempo diferente, aonde $\tau = r_Pt$:

$\frac{dp}{d\tau} = (1 - p) p - \frac{\alpha _P}{r_P}Ap - \frac{aK_m}{r_P}pm$

$\frac{dm}{d\tau} = \frac{r_M}{r_P} (1 - m) m - \frac{\alpha _M}{r_M}Am - \frac{baK_P}{r_M}pm$

A partir dessa transformao, podemos ainda renomear algumas constantes:

  • $\rho = \frac{r_M}{r_P}$
  • $c_P = \frac{\alpha _P}{r_P}$
  • $d_P = \frac{aK_m}{r_P}$
  • $c_M = \frac{\alpha _M}{r_M}$
  • $d_M = \frac{baK_P}{r_M}$

Que enfim, resulta nas seguintes equaes:

$\frac{dp}{d\tau} = p[(1 - p) - c_PA - d_Pm]$

$\frac{dm}{d\tau} = m[\rho (1 - m) - c_MA - d_Mp]$

Com as equaes da dinmica enfim estabelecidas, podemos analisar o comportamento do mesopredador e da presa procura do efeito descrito anteriormente.

Equilbrio

De acordo com o artigo, o comportamento do sistema procura desse efeito analisado no equilbrio, mas especificamente no ponto de coexistncia equlibrada entre os dois atores. Analisando as equaes acima, isso acontece quando:

$\frac{dp}{d\tau} = 0$

$\frac{dm}{d\tau} = 0$

Ou seja, quando:

$0 = p[(1 - p) - c_PA - d_Pm]$

$0 = m[\rho (1 - m) - c_MA - d_Mp]$

Nessa situao, h quatro possibilidades:

  1. $p = 0$, $m = 0$
  2. $p = 0$, $m \neq 0$
  3. $p \neq 0$, $m = 0$
  4. $p \neq 0$, $m \neq 0$

A primeira uma situao bvia de soluo do problema, visto que quando a densidade populacional dos dois atores zero, no h dinmica no problema. Sendo assim, analisemos as outras situaes.

2. $p = 0$, $m \neq 0 $

Com $p = 0$ e $m \neq 0$, temos:

$0 = 0[(1 - 0) - c_PA - d_Pm_1^*]$

$0 = m_1^*[\rho (1 - m_1^*) - c_MA - d_M0]$

Que se desenvolve em:

$0 = \rho (1 - m_1^*) - c_M A$

$\rho m_1^* = \rho 1 - c_MA $

$ m_1^* = 1 - \frac{c_MA}{\rho }$

3. $p \neq 0$, $m = 0$

Com $p \neq 0$ e $m = 0$, temos:

$0 = p_2^*[(1 - p_2^*) - c_PA - d_P0]$

$0 = 0[\rho (1 - 0) - c_MA - d_Mp_2^*]$

Que se desenvolve em:

$0 = (1 - p_2^*) - c_PA $

$p_2^* = 1 - c_PA $

4. $p \neq 0$, $m \neq 0$

Como dito anteriormente, a situao de interesse o equilbrio de coexistncia entre os dois atores, ou seja, a situao 4.

$0 = p_3^*[(1 - p_3^*) - c_PA - d_Pm_3^*]$

$0 = 1 - p_3^* - c_P A -d_P m_3^*$

$p_3^* = 1 - c_P A - d_P m_3^* $

$0 = m_3^*[\rho (1 - m_3^*) - c_MA - d_Mp_3^*]$

$0 =\rho - \rho m_3^* - c_M A - d_M p_3^*$

$0 = \rho - \rho m_3^* - c_M A - d_M (1 - c_P A - d_P m_3^*)$

$0 = \rho - \rho m_3^* - c_M A + d_M c_P A + d_M d_P m_3^* - d_M$

$0 = \rho + m_3^*(d_M d_P - \rho) + ( d_M c_P - c_M ) A - d_M$

$m_3^*(d_M d_P - \rho) = -\rho - ( d_M c_P - c_M ) A + d_M$

Portanto:

$m_3^* = \frac{-\rho - ( d_M c_P - c_M ) A + d_M}{(d_M d_P - \rho)}$

$p_3^* = 1 - c_P A - d_P \frac{-\rho - ( d_M c_P - c_M ) A + d_M}{(d_M d_P - \rho)} $

Existem, ento, quatro situaes de equilbrio:

  1. $p_0^* = 0$, $m_0^* = 0$
  2. $p_1^* = 0$, $ m_1^* = 1 - \frac{c_MA}{\rho }$
  3. $p_2^* = 1 - c_PA $, $m_2^* = 0$
  4. $p_3^* = 1 - c_P A - d_P \frac{-\rho - ( d_M c_P - c_M ) A + d_M}{(d_M d_P - \rho)} $, $m_3^* = \frac{-\rho - ( d_M c_P - c_M ) A + d_M}{(d_M d_P - \rho)}$

Implementao

Implementemos ento, essas situaes, para quatro casos:

  1. $\alpha_p = 0.5$, $K_m = 0.75$
  2. $\alpha_p = 0.5$, $K_m = 0.25$
  3. $\alpha_p = 2.0$, $K_m = 0.5$
  4. $\alpha_p = 0.5$, $K_m = 2.0$

Os demais parmetros so:

  • $r_m = 1.0$
  • $r_p = 1.0$
  • $a = 1.0$
  • $b = 0.1$
  • $\alpha_m = 1.0$
  • $K_p = 2.0$

def m_func(rho, cm, A, dm, dp, cp):
    """Funo de clculo do valor de m no equilbrio de coexistncia."""
    return (-rho - (dm*cp - cm)*A + dm)/(dm*dp - rho)

def p_func(rho, cm, A, dm, dp, cp):
    """Funo de clculo do valor de p no equilbrio de coexistncia."""
    return 1 - cp*A - dp*(-rho - (dm*cp - cm)*A + dm)/(dm*dp - rho)

def iterator(rho, cm, A, dm, dp, cp):
    """Itera a simulao."""
    p = np.zeros(A.shape)
    m = np.zeros(A.shape)

    # Inicializao dos valores
    p[0] = p_func(rho, cm, A[0], dm, dp, cp)
    m[0] = m_func(rho, cm, A[0], dm, dp, cp)
    
    # Clculo para todo A
    for i in range(1, A.shape[0]):

        p_ = 0
        m_ = 0

        if m[i - 1] <= 0.0:
            # Caso o mesopredador se extingua, 
            # o equilbrio agora ser com apenas p, 
            # obedecendo p_2*
            p_ = (1 - cp*A[i])
        else:
            # Se no, obedece p_3* e m_3*
            p_ = p_func(rho, cm, A[i], dm, dp, cp)
            

        if p[i - 1] <= 0.0:
            # Caso a presa se extingua, 
            # o equilbrio agora ser com apenas m, 
            # obedecendo m_1*
            m_ = (1 - (cm/rho)*A[i])
        else:
            # Se no, obedece p_3* e m_3*
            m_ = m_func(rho, cm, A[i], dm, dp, cp)

        # Endereando populao negativa e 
        # atualizando os valores de p e m
        p[i] = p_ if p_ >= 0.0 else 0.0
        m[i] = m_ if m_ >= 0.0 else 0.0

    return p, m
