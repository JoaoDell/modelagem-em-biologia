Introduo

A Influenza em humanos, causada por trs tipos de vrus diferentes (A, B, C e seus subtipos), uma doena respiratria que acomete a espcie humana h milnios, com alguns dos primeiros registros de uma epidemia em 6000 BCE. Seus sintomas incluem febre aguda, tosse, garganta inflamada, dores no corpo e indisposio, com intensidade dos sintomas a depender do tipo de vrus infectado e de sua carga viral.

A Influenza j possui preveno por meio de vacinao, que realizada anualmente pelo Ministrio da Sade, e recomendada a todas as idades, apesar de perder eficincia em idosos.

Modelagem

Para modelar a Influenza, usaremos de base o modelo SIR (Suscetvel, Infectado, Removido), com algumas alteraes. A primeira alterao levar em conta a sazonalidade do vrus, que tende a incidir com maior frequncia em pocas de inverno, quando por diminuio da temperatura, uma maior quantidade de poluio e patgenos se assentam perto da superfcie, causando queda de imunidade e problemas respiratrios. Para levar a sazonalidade em conta, introduziremos uma dependncia temporal na tradicional varivel de transmisso, $\beta$, que contar com um comportamento gaussiano peridico do seguinte formato:

$$\beta (t) = Be^{-Csen(\omega t + \phi)^2} + D$$

Sendo cada parmetro:

  • $B$ : Amplitude da transmisso
  • $C$ : Rapidez da transmisso
  • $\omega$ : Frequncia anual
  • $\phi$ : Estado inicial da transmisso
  • $D$ : Valor base de transmisso

A segunda alterao ser a adio de uma taxa de perda de imunidade $\delta$ populao de removidos. Essa taxa levar em conta possveis mutaes dos vrus, um fenmeno extremamente comum, que podero levar ao escapamento dos mesmos dos mecanismos de defesa anteriormente desenvolvidos pelo corpo dos infectados.

Levando em conta ento essas alteraes, o modelo SIR para a Influenza se torna:

$$\frac{dS(t)}{dt} = -\beta (t) S I + \delta R$$

$$\frac{dI(t)}{dt} = \beta (t) S I - \gamma I$$

$$\frac{dR(t)}{dt} = \gamma I - \delta R$$

  • $S$ : Suscetveis
  • $I$ : Infectados
  • $R$ : Removidos
  • $\beta (t)$ : Taxa de transmisso sazonal do vrus
  • $\gamma$ : Taxa de recuperao dos infectados
  • $\delta$ : Taxa de mutaes que escapam a imunidade

possvel averiguar pelo modelo acima que estamos trabalhando com uma populao constante, ou seja $\frac{dS(t)}{dt} + \frac{dI(t)}{dt} + \frac{dR(t)}{dt} = 0$, que se traduz em $S + I + R = N$.

Condio para Epidemia

Para que uma epidemia acontea, $\frac{dI}{dt} > 0$, ou seja:

$$\beta (t) S I - \gamma I > 0$$

$$\beta (t) S I > \gamma I$$

$$ S > \frac{\gamma}{\beta (t)}$$

Portanto, se $S$ for maior que $\frac{\gamma}{\beta (t)}$, uma epidemia acontece.

Pontos fixos e estabilidade

Para analisar os pontos fixos, devemos analisar os pontos os quais as trs equaes diferenciais so iguals a zero:

$$\frac{dS(t)}{dt} = -\beta (t) S I + \delta R = 0$$

$$\frac{dI(t)}{dt} = \beta (t) S I - \gamma I = 0$$

$$\frac{dR(t)}{dt} = \gamma I - \delta R = 0$$

Para $\frac{dS(t)}{dt}$:

$$\frac{dS(t)}{dt} = -\beta (t) S I + \delta R = 0$$

$$-\beta (t) S^* I^* = -\delta R^*$$

$$ S^* = \frac{\delta R^*}{\beta (t)I^*}$$

Para $\frac{dI(t)}{dt}$:

$$\frac{dI(t)}{dt} = \beta (t) S^* I^* - \gamma I^* = 0$$

$$(\beta (t) S^* - \gamma) I^* = 0$$

Para $\frac{dR(t)}{dt}$:

$$\frac{dR(t)}{dt} = \gamma I - \delta R = 0$$

$$ \delta R^* = \gamma I^*$$

$$ R^* = \frac{\gamma I^*}{\delta}$$

A resoluo dessas trs equaes nos rende os seguintes resultados:

  • $ S^* = \frac{\gamma }{\beta (t)}$
  • $(\beta (t) S^* - \gamma) I^* = 0$
  • $ R^* = \frac{\gamma I^*}{\delta}$

Sendo assim, encontramos dois pontos:

  1. $(S^* = \frac{\gamma}{\beta (t)}, 0, 0)$

  2. $(S^* = \frac{\gamma}{\beta (t)}, I^*, \frac{\gamma I^*}{\delta})$

Percebe-se que $S^* = \frac{\gamma}{\beta (t)}$ um valor fixo de $S$ caracterstico dos dois pontos de equilbrio possveis. Com relao ao primeiro, temos $I^* = 0$, portanto, no h espalhamento da doena. J, no segundo, h equilbrio se, para qualquer valor de $I*$, $R = \frac{\gamma I^*}{\delta}$.

from scipy.integrate import odeint

def SIRt(y, t, beta, gamma, delta, *args):
    """Funo que modela o comportamento do SIR com parmetro beta com variao temporal.
    """
    Sn, In, Rn = y
    dS = - beta(t, *args)*Sn*In + delta*Rn 
    dI = beta(t, *args)*Sn*In -gamma*In 
    dR = gamma*In - delta*Rn 
    return dS, dI, dR
